Chapter 2: Your First Program

Let's start with the traditional "Hello, World!" program. This simple example demonstrates the basic structure of a program:

fun main() { println("Hello, World!") println("Welcome to the world of programming!") }

This Kotlin code does the following:

  1. Defines a main function - the entry point of the program
  2. Prints "Hello, World!" to the console
  3. Prints a welcome message
